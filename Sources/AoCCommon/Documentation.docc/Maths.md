# Maths

Small mathematical utilities that support common Advent of Code problems.

These helpers provide functionality such as modular arithmetic, the extended
Euclidean algorithm, and solving simple Diophantine systems.

## Topics

### Modular Arithmetic
- ``mod(_:_:)``

### Number Theory
- ``extendedEuclid(_:_:)``
- ``diophantineEEA(ax:bx:ay:by:cx:cy:)``
